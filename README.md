## SMGAE: A novel self-supervised masked graph autoencoder for identifying cancer driver genes.

# Dependencies
Higher versions should be also available.

+ numpy==1.21.6
+ torch==1.12.1+cu102
+ torch-cluster==1.6.0
+ torch_geometric>=2.4.0
+ torch-scatter==2.0.9
+ torch-sparse==0.6.14
+ scipy==1.7.3
+ texttable==1.6.2
+ CUDA 10.2
+ CUDNN 7.6.0

## Clone this project
```bash
git clone https://github.com/liwenjun111/SMGAE.git
cd SMGAE
```

## Installation

```bash
pip install -r requirements.txt
```

## Running SMGAE

```bash
python main_transductive.py 
```
